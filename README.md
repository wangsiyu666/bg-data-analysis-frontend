图表：使用 ECharts，动画时长0.5s。

确认框：Element Plus MessageBox.confirm。

五、通用补充说明

原型中的数值（如5,134万、12,584,203等）在初期可作为Mock数据写死在前端，但所有交互接口应保留，后续由后端替换真实数据。

2. 响应式与适配
最小宽度1200px，低于此宽度显示横向滚动条。
卡片内文字在小屏时可换行。
3. 错误处理
所有网络请求失败时，显示统一错误提示，并提供重试按钮（在图表区域）。
长时间加载（>5秒）显示“数据加载中，请稍候”。

图表使用ECharts的notMerge和lazyUpdate。
表格分页使用后端分页，避免一次性加载大量数据。
